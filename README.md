# Calculadora IMC

Projeto simples de calculadora de IMC (Indice de Massa Corporal) feito em
HTML, CSS e JavaScript puro. Permite informar peso e altura e exibe a
classificacao do IMC.

## Funcionalidades

- Entrada de peso (kg) e altura (m)
- Calculo do IMC
- Exibicao da mensagem de classificacao
- Interface responsiva e estilizada

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha peso e altura.
3. Clique em **Calcular**.

## Estrutura do projeto

- `index.html` - Estrutura da pagina
- `style.css` - Estilos da interface
- `script.js` - Logica do calculo

## Observacoes

- Use ponto para decimais (ex: `1.75`).
